plugin-prestashop
=================

LiqPay plugin for prestashop CMS

#### Підтримувана версія - Prestashop 1.8 ####

#### Для встановлення модуля скопіюйте його (liqpaypaqments) в каталог: ####

```
<PresatShop>/modules/
```

#### Потім в панелі адміністратора встановіть і налаштуйте його: ####

```
Modules > Modules > Liqpay натиснути [Install]
```


#### Для початку прийому платежів вам потрібно: ####
    - Зареєструватись на https://www.liqpay.ua
    - Створити магазин використовуючи майстер установки (https://www.liqpay.ua/admin/business)
    - Отримайте "Публічний ключ" і "Приватний ключ" в розділі "Мої магазини" (https://www.liqpay.ua/admin/business)

#### Що нового в версії 3.1 ####
    - Підтримка Prestashop 1.8
    - Підтримка мови української
    - Підтримка мови англійської
    - Підтримка валют UAH, USD, EUR
    - Розширена підтримка статусів платежу згідно https://www.liqpay.ua/doc/api/callback


#### Особливості ####
В наступній версії буде додано налаштування server_url та result_url.
